КЛИЕНТ (curl/браузер)
        │
        │ HTTP запрос
        ▼
┌───────────────┐
│  HTTP АГЕНТ   │  ← Стоит на порту 8080, принимает запросы
│ (секретарь)   │
└───────┬───────┘
        │
        │ Отправляет сообщение (msg_create_record)
        ▼
┌───────────────┐
│   DB АГЕНТ    │  ← Обрабатывает данные (пока просто выводит)
│ (начальник)   │
└───────────────┘






Клиент (curl) → POST /api/v1/records
       ↓
HTTP агент (секретарь) принимает запрос
       ↓
Генерирует UUID и создаёт сообщение msg_create_record
       ↓
Отправляет сообщение DB агенту через SObjectizer
       ↓
DB агент (начальник) получает сообщение
       ↓
Сохраняет данные в SQLite3 (файл records.db)
       ↓
HTTP агент возвращает ответ клиенту: {"status": "created", "id": "..."}



HTTP агент → отправляет msg_get_records → DB агент → читает БД → возвращает данные → HTTP агент → отдаёт JSON
